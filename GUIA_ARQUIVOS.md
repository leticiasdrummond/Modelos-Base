# Guia dos arquivos do pacote

## Para começar

Leia `README_DADOS_BRASIL.md` e execute `python main_execucao.py`.

## Para pesquisadores

Use `REVISAO_DADOS_NORMATIVOS_BRASIL.md` como registro de premissas e `DISSERTACAO_ANALISE_COMPLETA.md` como esqueleto textual. Ajuste as seções para o problema de pesquisa, inclua resultados reais e revise as referências conforme ABNT.

## Para desenvolvedores

O arquivo `modelo_eletropostos.py` contém funções pequenas e testáveis. O script `main_execucao.py` demonstra uma execução sem solver externo e grava JSON para inspeção.

## Para gestores

O `RESUMO_EXECUTIVO.md` apresenta as mensagens principais e cuidados de validação antes de usar os números em decisões de investimento.
